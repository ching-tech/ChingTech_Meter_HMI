# UI 介面規格

## 框架
- **NiceGUI** 以 Native 桌面模式運行
- 深色主題 (`dark=True`)
- 預設視窗大小: 1200x850
- 支援 F5 重新整理
- Master port=8080, Slave port=8081

## 版面配置

### 頂部狀態列 (Header Card)
```
┌────────────────────────────────────────────────────────────────────┐
│ 🏥 擎添耳溫槍探頭套檢測系統 v2.1.0 [MASTER/SLAVE]                │
│ 目前批次: YYYYMMDDHHMMSS.csv [換批] | 測試週期: N | 系統狀態: 運行中│
│                                        PLC: ● 網路: ● ⚙          │
└────────────────────────────────────────────────────────────────────┘
```
- 測試週期: 僅 Master 顯示

### 警告橫幅 (Alert Container)
- 預設隱藏
- 藍芽斷線時顯示紅色背景 + 閃爍效果 (0.5s 間隔)
- 顯示內容: `⚠ [HH:MM:SS] CHx 藍芽斷線!`
- 「確認」按鈕停止閃爍

### 主要區域 (Row Layout)

#### Master 模式
```
┌─────────────────────┐ ┌─────────────────────┐ ┌──────────┐ ┌────────┐
│ 本機通道 (CH11,9,7, │ │ Slave通道 (CH12,10, │ │ 控制面板 │ │系統 Log│
│  5,3,1)  [藍框]     │ │  8,6,4,2) [橘框]    │ │          │ │        │
│ CH BT 耳套 空槍 量測│ │ CH BT 耳套 空槍 量測│ │          │ │        │
│  誤差 狀態 OK NG    │ │  誤差 狀態 OK NG    │ │          │ │        │
└─────────────────────┘ └─────────────────────┘ └──────────┘ └────────┘
```

#### Slave 模式
```
┌─────────────────────┐ ┌──────────┐ ┌────────┐
│ 本機通道 (CH12,10,  │ │ 控制面板 │ │系統 Log│
│  8,6,4,2) [橘框]    │ │          │ │        │
│ CH BT 耳套 空槍 量測│ │          │ │        │
│  誤差 狀態           │ │          │ │        │
│ (無 OK/NG 欄位)     │ │          │ │        │
└─────────────────────┘ └──────────┘ └────────┘
```
- Slave 不顯示 OK/NG 計數欄位 (由 PLC 管理，只有 Master 有 PLC)

#### 通道卡片欄位
| 欄位 | 寬度 | 說明 | Master | Slave |
|------|------|------|--------|-------|
| CH | w-16 | 通道名稱 + 停用徽章 | Y | Y |
| BT | w-8 | 藍芽連線狀態圖示 | Y | Y |
| 耳套 | w-12 | 有/無/-- | Y | Y |
| 空槍值 | w-24 | cyan 文字，%.2f | Y | Y |
| 量測值 | w-24 | yellow 文字，%.2f | Y | Y |
| 誤差 | w-24 | white 文字，%.2f | Y | Y |
| 狀態 | w-24 | 燈號 + PASS/FAIL/WAIT | Y | Y |
| OK | w-14 | 綠色計數 | Y | **N** |
| NG | w-14 | 紅色計數 | Y | **N** |

#### Slave UI 即時顯示
- 藍芽收到資料時根據量測狀態決定:
  - `WAITING_EMPTY` / `EMPTY_DONE` → 顯示在 `empty_display`
  - 其他狀態 → 顯示在 `temp_display`

#### 最右側: 系統 Log
- `ui.log` 元件，最多 100 行
- 等寬字體，灰色文字
- 高度 600px

### 右側抽屜: 進階設定 (Settings Drawer)
- 點擊 ⚙ 按鈕開啟
- 寬度 320px
- 區塊:
  - 系統設定 (誤差上下限、運行模式)
  - 時序設定 (5 個延遲參數)
  - PLC 設定 (IP/Port)
  - 藍芽設定 (6 組 MAC 位址、重連間隔、超時)
  - 通道啟用 (12 個開關，6x2 網格)
- 按鈕:
  - **儲存進階設定** (藍色): 存檔，部分參數即時生效 (誤差值、通道啟用)
  - **更新資料 (即時套用)** (綠色): 存檔 + 所有參數即時套用到執行中管理器

## 狀態指示

### 通道狀態燈
| 顏色 | 意義 |
|------|------|
| 綠色 (green) | PASS |
| 紅色 (red) | FAIL |
| 灰色 (grey) | WAIT (等待) |

### 藍芽狀態圖示
| 顏色 | 意義 |
|------|------|
| 藍色 (blue) | 已連線 |
| 黃色 (yellow) | 連線中 |
| 紅色 (red) | 錯誤 |
| 灰色 (gray) | 未連線 |

### PLC 連線指示
| 顏色 | 意義 |
|------|------|
| 綠色 | 已連線 |
| 黃色 | 連線中 |
| 紅色 | 未連線/錯誤 |

### 停用通道
- 半透明顯示 (`opacity-40`)
- 紅色「停用」徽章
