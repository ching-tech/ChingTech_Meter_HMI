解決更換 Dongle 後藍牙設備無法移除的問題

目錄 (依推薦順序嘗試)：
  • 方法一：透過「依容器列出裝置」強制刪除 (專治清單卡死)
  • 方法二：拔除 Dongle 進行「無硬體全盤清掃」
  • 方法三：顯示隱藏裝置並解除安裝
  • 方法四：透過登錄檔 (Regedit) 強制拔除配對
  • 方法五：飛航模式與服務重啟法

方法一：透過「依容器列出裝置」強制刪除

專門對付在「裝置和印表機」清單中看得到，卻一直顯示移除失敗的狀況。

  1. 打開「裝置管理員」。
  2. 點擊上方選單的「檢視」，選擇「依容器列出裝置」。
  3. 在列表中尋找你卡住的設備名稱並展開。
  4. 對著它（或底下的子項目）按右鍵，選擇「解除安裝裝置」。

方法二：拔除 Dongle 進行「無硬體全盤清掃」

  1. 拔下目前插在新 Dongle。
  2. 打開「裝置管理員」，點選「檢視」 > 「顯示隱藏的裝置」。
  3. 展開「藍牙」分類。將裡面的所有項目全部點右鍵選擇「解除安裝裝置」。
  4. 重新開機（開機前不要插上 Dongle）。
  5. 開機後確認設備已消失，再插回新 Dongle。

方法三：顯示隱藏裝置並解除安裝

  1. 打開「裝置管理員」。
  2. 點擊「檢視」 > 「顯示隱藏的裝置」。
  3. 展開「藍牙」分類，找到半透明的卡住設備，按右鍵選擇「解除安裝裝置」。

方法四：透過登錄檔 (Regedit) 強制拔除配對

  1. 按下 Win + R，輸入 regedit 並按 Enter。
  2. 導覽至：
    Computer\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\BTHPORT\Parameters\Devices
  3. 展開 Devices 資料夾，點開裡面的 12 碼資料夾查看 Name 數值。
  4. 確認設備後，對著該 12 碼資料夾按右鍵選擇「刪除」。重新開機。

方法五：飛航模式與服務重啟法

  1. 飛航模式法：開啟「飛航模式」，趁藍牙強制關閉時嘗試「移除裝置」。
  2. 重啟服務法：按下 Win + R 輸入 services.msc，找到「藍牙支援服務」，按右鍵選擇「重新啟動」。
